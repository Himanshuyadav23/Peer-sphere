# Quick APK Build Guide

## Fast Track (5 Steps)

1. **Install EAS CLI:**
   ```bash
   npm install -g eas-cli
   ```

2. **Login:**
   ```bash
   cd PeerSphereMobile
   eas login
   ```

3. **Configure (first time only):**
   ```bash
   eas build:configure
   ```

4. **Build APK:**
   ```bash
   npm run build:apk
   ```

5. **Download & Share:**
   - Wait 10-20 minutes
   - Download from Expo dashboard or terminal link
   - Share APK file with others
